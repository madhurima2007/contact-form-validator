# Advanced Contact Form Validation

## Overview

A modern responsive contact form built using:

- HTML5
- CSS3
- JavaScript (ES6)

The project performs client-side validation using JavaScript and Regex while providing user-friendly feedback.

---

## Features

✔ Glassmorphism UI

✔ Responsive Design

✔ Regex Email Validation

✔ Real-Time Character Counter

✔ Error Messages

✔ Success Message

✔ Prevent Invalid Submission

✔ Modern Animations

✔ Mobile Friendly

---

## Validation Rules

### Name

- Cannot be empty
- Minimum 3 characters

### Email

- Cannot be empty
- Must follow valid email format

### Message

- Cannot be empty
- Minimum 10 characters
